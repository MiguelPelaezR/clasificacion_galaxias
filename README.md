# clasificacion_galaxias
Programa de machine-learning y CNN para predecir, a través de parámetros físicos e imágenes, si una observación es de dos galaxias fusionándose o no


Las imagenes de las galaxias no se pueden añadir a GitHub por ser un archivo muy pesado, se pueden descargar en:
https://zenodo.org/records/15106473

Todos los datos están sacados de esta web.
